# Personal Finance Coach Skill

A specialized AI Edge Gallery skill for practical budgeting, savings planning, debt payoff strategy, and monthly cashflow improvement.

## Overview
This skill helps users make clear, numbers-driven financial decisions using simple frameworks. It supports users who want to reduce financial stress, improve savings consistency, and pay down debt with realistic plans.

## Installation

### Quick Install (Android)
1. Open the **Google AI Edge Gallery** app.
2. Select the **Agent Skills** option from the main menu.
3. Select the **Gemma-4-E2B-it** model by pressing the **Try it** button and wait for everything to load.
4. Press the **Skills Icon** at the bottom of the GUI.
5. Press the **Plus (+)** icon and select **Load skill from URL**.
6. Enter the following URL:
   `https://raw.githubusercontent.com/StrinGhost/gemma-skills/main/skills/personal-finance-coach/SKILL.md`
7. Select **Add**.

### Quick Install (iPhone / iOS)
1. Open the [SKILL.md URL](https://raw.githubusercontent.com/StrinGhost/gemma-skills/main/skills/personal-finance-coach/SKILL.md) in Safari.
2. Tap the **Share** icon and select **Save to Files**. Save it to your **Downloads** folder.
3. Open the **Google AI Edge Gallery** app.
4. Select the **Gemma-4-E2B-it** model and press **Try it**.
5. Press the **Skills Icon** at the bottom of the GUI.
6. Press the **Plus (+)** icon and select **Load skill from Local**.
7. Navigate to your **Downloads** folder and select the `SKILL.md` file you just saved.

### Manual Install
1. Create a folder named `personal-finance-coach` in your local AI Edge Gallery skills directory.
2. Save the `SKILL.md` and `README.md` files into that folder.
3. Open the **Google AI Edge Gallery** app.
4. Navigate to **Agent Skills** and tap the **Refresh** icon to activate.

## Usage
Trigger this skill for personal finance planning questions, such as:
- "Help me build a monthly budget from this income and expense list."
- "Should I prioritize emergency savings or credit card debt first?"
- "Give me a 30-day cashflow stabilization plan."

## Important Notice
This skill provides general personal finance education, not regulated financial, tax, or legal advice. Users should verify major financial decisions with qualified professionals.
